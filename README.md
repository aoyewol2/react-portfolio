## Setup

1. Install dependencies

   ```sh
   npm install
   ```

2. Start the development server

   ```sh
   npm start
   ```

## Build and run for production

1. Generate a full static production build

   ```sh
   npm run build
   ```
2. Preview the site as it will appear once deployed

   ```sh
   npm run serve
   ```
